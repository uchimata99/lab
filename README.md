# המעבדה — איך מעלים לגיט ומתחזקים

## חלק א׳ — להעלות ולפרסם (בלי קלוד קוד)
1. היכנסי ל‑github.com וצרי מאגר חדש (Public), למשל בשם `lab`.
2. העלי למאגר את הקבצים (Add file ← Upload files):
   - `index.html`
   - `CLAUDE.md`
   - `README.md`
   - `PROMPT.md`
3. במאגר: Settings ← Pages. תחת Branch בחרי `main` ותיקייה `/ (root)`, ולחצי Save.
4. המתיני כדקה. תופיע כתובת בסגנון:
   `https://<שם-המשתמש>.github.io/lab/`
5. פתחי את הכתובת בטאבלט בכרום או ב‑Samsung Internet. אפשר גם תפריט ← "הוסף למסך הבית".

## חלק ב׳ — לעבוד עם קלוד קוד (לעדכונים)
6. התקיני את קלוד קוד פעם אחת.
   - מק/לינוקס: `curl -fsSL https://claude.ai/install.sh | bash`
   - Windows PowerShell: `irm https://claude.ai/install.ps1 | iex`
7. שכפלי את המאגר למחשב:
   `git clone https://github.com/<שם-המשתמש>/lab.git`
8. כנסי לתיקייה והפעילי:
   - `cd lab`
   - `claude`
   (בפעם הראשונה תתבקשי להתחבר בדפדפן.)
9. הדביקי את הפרומפט מהקובץ `PROMPT.md`, או כתבי בעברית רגילה מה לשנות — למשל: "הוסיפי שמות עבריים לכל היסודות בטבלה".
10. כשמרוצה, כתבי: "תעשי commit ותדחפי". GitHub Pages יתעדכן לבד תוך דקה, והאפליקציה שעל מסך הבית תקבל את הגרסה החדשה ברענון.

מכאן והלאה: לעדכן = לתאר שינוי לקלוד קוד, ואז "commit ודחיפה".
