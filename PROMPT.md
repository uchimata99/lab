# פרומפט פתיחה לקלוד קוד

הפעילי את קלוד קוד בתוך תיקיית הפרויקט (`claude`), ואז העתיקי והדביקי את הטקסט שבין הקווים:

---------------------------------------------

שלום. בתיקייה הזאת יש קובץ `index.html` — אפליקציית לימוד אינטראקטיבית בעברית בשם "המעבדה", לילדה מחוננת.

ראשית, קראי את הקובץ `CLAUDE.md` כדי להבין את הפרויקט ואת העקרונות שאסור לשבור.

המשימה הראשונה — להעלות את האפליקציה לגיט ולפרסם אותה:
1. אתחלי מאגר Git בתיקייה (`git init`) וצרי קובץ `.gitignore` בסיסי.
2. הוסיפי את כל הקבצים ועשי commit ראשון בשם: "המעבדה — גרסה ראשונה".
3. הסבירי לי בעברית, צעד‑צעד ועם הפקודות המדויקות, איך לחבר את המאגר ל‑GitHub ולהפעיל GitHub Pages, כך שהאפליקציה תהיה זמינה בכתובת אינטרנט.

מכאן והלאה, בכל פעם שאבקש שינוי:
- שמרי תמיד על העקרונות שמופיעים ב‑`CLAUDE.md` (קובץ יחיד, עברית בלבד, פס שבר לחילוק, ערכת הנושא והגופנים).
- לפני כל commit הריצי את שתי בדיקות האימות: בדיקת תחביר (`node --check`) ובדיקת ריצה שמדמה את כל השיעורים.
- דברי איתי בעברית, והסבירי כל שינוי לפני שאת מבצעת אותו.

---------------------------------------------
